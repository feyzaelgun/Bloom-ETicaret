Bloom – Çiçek Satış Web Sitesi

Bloom, kullanıcıların çiçekleri inceleyip sipariş sürecine yönlendirilebildiği, modern ve responsive tasarıma sahip bir web arayüzüdür.

Özellikler
Anasayfa üzerinden ürünlere hızlı erişim
Ürünler, Hakkımızda ve İletişim sayfalarına yönlendirme
Görsel ağırlıklı slider (anasayfa banner)
Mobil uyumlu (responsive) tasarım
Sepet altyapısı için temel arayüz

Sayfalar
index.php – Anasayfa
shop.php – Ürünler
about.php – Hakkımızda
contact.php – İletişim

Teknolojiler
HTML5
CSS3
Bootstrap
JavaScript / jQuery
